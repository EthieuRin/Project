# Git Workflow for BBIT Enterprise Project

1. Initialize the repository
   git init

2. Add all project files (PHP, HTML, CSS, etc.)
   git add register.php mail.php vendor/ database.sql

3. Commit the changes with a message
   git commit -m "Initial commit with register.php, mail.php and database setup"

4. Add a remote repository (https://github.com/EthieuRin/Project.git)
   git remote add origin https://github.com/EthieuRin/Project.git

5. Push the changes to the remote repository
   git push -u origin main

6. For future changes:
   git add .
   git commit -m "Describe the changes made"
   git push
